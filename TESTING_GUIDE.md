# Zelda's Daughter — Гайд по тестированию на iPhone

Пошаговая инструкция для ручного тестирования всех механик на устройстве.
Ориентировочное время полного прогона: **40-50 минут**.

Устройство в вертикальной ориентации. Управление одной рукой.

---

## 0. Перед запуском

- [ ] Билд установлен через Xcode / TestFlight
- [ ] iPhone разблокирован, режим "Не беспокоить" включён
- [ ] Яркость экрана на максимум (для проверки дня/ночи)
- [ ] Звук включён (проверка ambient, шагов, боя)
- [ ] Секундомер под рукой (для замера FPS-провалов и таймингов)

---

## 1. Запуск и первые секунды (~1 мин)

**Что проверяем:** приложение стартует, нет крашей, ориентация верная.

- [ ] Приложение запускается без краша
- [ ] Экран вертикальный (portrait), не переворачивается при наклоне
- [ ] Нет розового экрана (шейдеры URP корректны)
- [ ] Персонаж виден на экране (изометрическая камера, угол ~30-45 градусов)
- [ ] Сцена загружена: видна поляна, дорога, деревья вокруг

---

## 2. Движение (Этап 1) — ~2 мин

**Что проверяем:** свайп-управление, анимации, камера, реакция окружения.

- [ ] Поставить палец на экран и вести — персонаж двигается в направлении свайпа
- [ ] Короткий свайп (палец близко к точке начала) — персонаж идёт шагом (анимация walk)
- [ ] Длинный свайп (палец далеко от точки начала) — персонаж бежит (анимация run)
- [ ] Отпустить палец — персонаж плавно останавливается (не резко, есть торможение)
- [ ] Персонаж поворачивается в сторону движения (плавный Slerp, не телепортация)
- [ ] Камера плавно следует за персонажем (без рывков)
- [ ] Камера не улетает за границы карты при движении к краю региона
- [ ] Пройти через кусты/траву — они покачиваются от прохода
- [ ] Персонаж не проваливается под землю нигде на карте
- [ ] Персонаж не проходит сквозь деревья, скалы, здания (коллайдеры)

**Баг-маркеры:** задержка ввода >0.3 сек, дёрганая камера, проход сквозь стены.

---

## 3. Онбординг (Этап 1) — ~1 мин

**Что проверяем:** неблокирующие подсказки появляются и исчезают.

- [ ] При пробуждении — подсказка свайпа (иконка пальца/стрелки)
- [ ] Подсказка НЕ блокирует управление — можно двигаться поверх неё
- [ ] После первого свайпа/движения — подсказка исчезает
- [ ] При приближении к первому интерактивному объекту — подсказка тапа
- [ ] После первого тапа — подсказка тапа исчезает

---

## 4. Взаимодействие с миром (Этап 2) — ~3 мин

**Что проверяем:** тап по объектам, подбор, добыча, осмотр.

- [ ] Подойти к лежащему предмету (палка/камень на земле) — предмет подсвечивается при приближении (emission)
- [ ] Тап по предмету — персонаж подходит к нему (автодвижение) и поднимает (анимация PickUp)
- [ ] Реплика в облачке над головой при подборе ("Палка... сгодится" или аналог)
- [ ] Облачко исчезает через несколько секунд
- [ ] Тап по дереву — персонаж подходит и бьёт (анимация атаки). После нескольких ударов — выпадают палки/дрова
- [ ] Дерево визуально реагирует на удары (качается, уменьшается)
- [ ] Тап по камню-ресурсу — аналогично, выпадают камни
- [ ] Тап по кусту — выпадают ягоды/ветки
- [ ] Тап по необычному объекту (камень-декор, бочка) — реплика персонажа (осмотр через Inspectable)
- [ ] Подобрать 3-5 разных предметов — все добавляются в инвентарь

**Баг-маркеры:** персонаж не подходит к объекту, предмет не подбирается, реплика не появляется.

---

## 5. Пролог: крестьянин (Этап 2) — ~1 мин

**Что проверяем:** NPC-взаимодействие через иконки.

- [ ] Пойти по дороге (~15 сек ходьбы) — найти крестьянина в поле/у хижины
- [ ] Тап по крестьянину — персонаж подходит, крестьянин поворачивается
- [ ] Крестьянин показывает иконки над головой (жест приветствия, стрелка к городу)
- [ ] Появляется панель ответов иконками (2-3 пиктограммы)
- [ ] Тап по иконке-ответу — крестьянин реагирует

---

## 6. Инвентарь (Этап 3) — ~3 мин

**Что проверяем:** лонг-пресс, радиальное меню, инвентарь, вес.

- [ ] Лонг-пресс на персонаже (палец на персонаже, держать >=0.5 сек, не двигать)
- [ ] Появляется кольцо-таймер вокруг персонажа (заполняется за 0.5 сек)
- [ ] Открывается радиальное меню с секторами (инвентарь, карта, блокнот)
- [ ] Игра на паузе (Time.timeScale = 0) пока меню открыто
- [ ] Вести палец к сектору "Инвентарь" и отпустить — открывается панель инвентаря
- [ ] Видны подобранные предметы в слотах (иконки, стаки)
- [ ] Тап по предмету — появляется описание (название + текст от лица героя)
- [ ] Закрыть инвентарь — игра снимается с паузы

**Проверка веса:**
- [ ] Набрать много предметов (15-20 камней/палок) — персонаж замедляется
- [ ] Реплика о перегрузе ("Тяжело..." или аналог)
- [ ] Выбросить часть предметов — скорость восстанавливается

**Важно:** лонг-пресс должен срабатывать ТОЛЬКО на персонаже. Лонг-пресс на пустом месте — ничего. Если палец сдвинулся на >20px — это свайп, не лонг-пресс.

---

## 7. Крафт (Этап 3) — ~5 мин

**Что проверяем:** drag & drop крафт, рецепты, размещение в мире.

### 7.1 Полевой крафт
- [ ] Открыть инвентарь. Лонг-пресс на предмете (~0.3 сек) — предмет "отлипает" и следует за пальцем
- [ ] Перетащить палку на ткань (или другую валидную пару) — крафт факела
- [ ] Эффект успешного крафта (визуал + реплика)
- [ ] Новый предмет появляется в инвентаре
- [ ] Перетащить два несовместимых предмета — реплика неудачи ("Это не сработает")
- [ ] Drag предмет за пределы панели инвентаря — переход в режим размещения (закрытие инвентаря, призрак предмета в мире)
- [ ] Отпустить на валидном месте — предмет размещён в мире
- [ ] Отпустить на невалидном месте (стена, воздух) — призрак красный, предмет возвращается в инвентарь

### 7.2 Рецепты для проверки
| Ингредиент A | Ингредиент B | Результат |
|---|---|---|
| Палка | Ткань | Факел |
| Нож | Палка | Заострённая палка |
| Топор | Дрова | Доски |
| Ткань | Травы | Бинт |
| Палка | Ткань | Шина |
| Особая трава | Жир | Мазь от ожогов |
| Специальные травы | — | Антидот |

### 7.3 Костёр
- [ ] Разместить дрова в мире (drag из инвентаря)
- [ ] Перетащить кремень на дрова в мире — костёр загорается
- [ ] Костёр горит, даёт свет
- [ ] Стоять рядом с костром — зона отдыха (ускоренное восстановление)

### 7.4 Логические цепочки
- [ ] Поднести факел к костру (drag из инвентаря на костёр) — факел загорается
- [ ] При удержании предмета рядом с подходящим объектом — реплика-подсказка

---

## 8. Боевая система (Этап 4) — ~5 мин

**Что проверяем:** удары, враги, урон, раны, нокаут.

### 8.1 Бой с кабаном
- [ ] Найти зону спавна кабанов (южнее стартовой поляны, искать по местности)
- [ ] Кабан бродит (Idle состояние, анимация)
- [ ] Тап по кабану — персонаж подходит и наносит удар
- [ ] Каждый тап = один удар (нет автоатаки)
- [ ] Кабан агрится, бежит на игрока (Chase), замахивается (визуальный сигнал)
- [ ] Есть окно для уклонения: увести палец, персонаж отходит
- [ ] Кабан бьёт головой — может нанести ПЕРЕЛОМ
- [ ] При переломе: персонаж хромает, замедление движения и атак
- [ ] Убить кабана — анимация смерти, туша остаётся в мире
- [ ] Тап по туше без ножа — минимальный лут (клык/перо)
- [ ] Тап по туше С ножом в инвентаре — полная разделка (шкура, мясо, кости)

### 8.2 Бой с волком
- [ ] Найти зону волков (дальше от старта, лес)
- [ ] Волк патрулирует, замечает игрока, подкрадывается
- [ ] Бросок волка — наносит КОЛОТУЮ РАНУ (кровотечение)
- [ ] При кровотечении: визуал (кровь на одежде), медленная потеря HP
- [ ] Реплика о состоянии ("Ничего страшного" или "Плохо дело...")
- [ ] Волк опаснее кабана (больше урон, быстрее)

### 8.3 Раны и лечение
- [ ] Перелом лечится ШИНОЙ (крафт: палка + ткань)
- [ ] Кровотечение лечится БИНТОМ (крафт: ткань + травы)
- [ ] Drag лекарство из инвентаря на персонажа — применение
- [ ] Без лечения раны заживают медленно сами
- [ ] У костра раны заживают быстрее

### 8.4 Нокаут
- [ ] Получить много урона — экран затемняется (проблески)
- [ ] Пробуждение через несколько секунд
- [ ] После нокаута раны остаются, персонаж встаёт раненым
- [ ] Нокаут короткий (не раздражает)

### 8.5 Оружие
- [ ] В инвентаре экипировать палку / меч (если скрафтил) — появляется в руке персонажа
- [ ] Разное оружие — разная скорость и урон (палка слабая и быстрая, меч сильный)

**Баг-маркеры:** враг не агрится, удар не наносит урон, нокаут зацикливается, рана не снимается лечением.

---

## 9. Еда и голод (Этап 4) — ~2 мин

**Что проверяем:** механика еды, скрытый голод.

- [ ] Открыть инвентарь, найти ягоды (или другую еду)
- [ ] Drag ягоду на персонажа — анимация поедания
- [ ] Слабый хил + утоление голода
- [ ] Долго не есть (5-10 мин игрового времени) — реплика о голоде
- [ ] При долгом голоде — замедление, деградация характеристик

---

## 10. Прогрессия навыков (Этап 5) — ~3 мин

**Что проверяем:** скрытый рост навыков, обратная связь.

- [ ] Подраться с 3-5 врагами — через какое-то время реплика типа "Удары стали увереннее..."
- [ ] Много ходить (особенно с перегрузом) — реплика про выносливость
- [ ] Удары становятся быстрее с опытом (скрытое — сравнить анимацию первого боя и пятого)
- [ ] Промахи тоже дают рост (медленнее)
- [ ] Первые удары неуклюжие (медленные), со временем — увереннее

**Примечание:** рост навыков скрытый, без цифр на экране. Проверять по ощущениям и репликам.

---

## 11. Дорога к городу — ~2 мин

**Что проверяем:** переход между зонами, навигация.

- [ ] Крестьянин показал направление к городу (иконка-стрелка)
- [ ] Пойти по дороге (видимые тайлы пути)
- [ ] По дороге могут встретиться враги
- [ ] Город виден на горизонте (здания Fantasy Town Kit)
- [ ] Вход в город — без загрузочного экрана (бесшовный)

---

## 12. Вода (Этап 1) — ~1 мин

**Что проверяем:** взаимодействие с водой.

- [ ] Найти реку/ручей
- [ ] Зайти в воду — персонаж замедляется
- [ ] Визуал: персонаж по пояс в воде
- [ ] На глубине — невидимая стена (нельзя плыть)
- [ ] Найти мост — можно пройти по мосту через реку

---

## 13. Город и NPC (Этап 6) — ~5 мин

**Что проверяем:** NPC, диалоги, языковая механика.

### 13.1 NPC и расписание
- [ ] В городе 7-8 NPC: торговец, кузнец, бармен, травница, стражник, жители
- [ ] NPC двигаются по расписанию (NavMeshAgent по вейпоинтам)
- [ ] Днём NPC на рабочих местах (торговец — рынок, кузнец — кузница)
- [ ] Вечером/ночью — NPC идут в таверну или домой

### 13.2 Диалоги и язык
- [ ] Тап по NPC — персонаж подходит, NPC поворачивается
- [ ] Облачко речи NPC: текст отображается КРАКОЗЯБРАМИ (руны Elder Futhark: ᚠᚢᚦᚨᚱ...)
- [ ] Typewriter-эффект (текст печатается побуквенно)
- [ ] Варианты ответа иконками (при низком понимании языка)
- [ ] Поговорить с 5-6 NPC подряд — часть слов начинает проявляться (скрэмбл уменьшается)
- [ ] Короткие слова раскрываются первыми

### 13.3 Разные NPC
- [ ] **Торговец** — много текста, нахваливает, можно открыть торговлю
- [ ] **Кузнец** — короткие фразы, по делу
- [ ] **Бармен** — болтливый, сплетни, слухи
- [ ] **Травница** — спокойная, объясняет про лечение
- [ ] **Стражник** — формальный, жалуется на службу
- [ ] У каждого NPC свой DialogueTree (разные ветви диалога)

---

## 14. Торговля (Этап 6) — ~3 мин

**Что проверяем:** бартер, интерфейс обмена.

- [ ] Тап по торговцу — диалог — выбрать "Показать товары" (или иконка при непонимании)
- [ ] Открывается UI торговли: два столбца (инвентарь игрока | инвентарь торговца)
- [ ] Зона обмена посередине
- [ ] Drag свой предмет в зону обмена + drag предмет торговца в зону обмена
- [ ] Индикатор баланса сделки (цвет: зелёный = выгодно, красный = невыгодно)
- [ ] Подтвердить обмен — предметы меняются
- [ ] НА СТАРТЕ доступен только бартер (герой не знает валюту)
- [ ] После роста понимания языка — разблокируются монеты
- [ ] Торговля затруднена при низком понимании (ограниченный UI)

---

## 15. Кузница (Этап 3) — ~3 мин

**Что проверяем:** станочный крафт.

- [ ] Найти плавильню и наковальню у кузнеца в городе
- [ ] Тап по плавильне — открывается UI станка
- [ ] Положить руду — запускается плавка (таймер)
- [ ] После плавки — забрать расплавленный металл
- [ ] Тап по наковальне — открывается UI станка
- [ ] Положить металл + палку — скрафтить меч
- [ ] Оружие крафтится ТОЛЬКО на наковальне (не в инвентаре)

**Рецепты наковальни:**
| Материалы | Результат |
|---|---|
| Металл + палка | Меч |
| Металл + короткая палка | Нож |
| Металл | Наконечник стрелы |

---

## 16. Квесты и блокнот (Этап 7) — ~3 мин

**Что проверяем:** получение квестов, блокнот, выполнение.

- [ ] Поговорить со стражником — квест "Убить кабанов" (boar_hunt)
- [ ] Поговорить с торговцем — квест "Принести шкуры волков" (wolf_hides)
- [ ] Лонг-пресс на персонаже — радиальное меню — сектор "Блокнот"
- [ ] Открывается блокнот — видны записи о квестах (от лица героя, неформально)
- [ ] Записи НЕ трекают прогресс (нет "3/5 кабанов") — просто текст
- [ ] Выполнить квест (убить кабанов) — вернуться к стражнику — NPC реагирует + награда
- [ ] В блокноте также появляются записи при крафте (рецепты) и лечении ран

**5 стартовых квестов:** wolf_hides, rare_herb, boar_hunt, ore, rumors.

---

## 17. Карта (Этап 7) — ~1 мин

**Что проверяем:** покупка карты, маркеры.

- [ ] Лонг-пресс — радиальное меню — сектор "Карта"
- [ ] Без покупки карты — пустая / сообщение что карты нет
- [ ] Купить карту у торговца (через бартер) — появляются контуры региона
- [ ] Позиция игрока отображается на карте (обновляется в реальном времени)
- [ ] Поговорить с NPC, который добавляет маркеры (add_map_marker) — метки на карте
- [ ] До 7 маркеров от разных NPC

---

## 18. День и ночь (Этап 1) — наблюдать в течение теста

**Что проверяем:** визуальный цикл, NPC-реакция.

- [ ] Свет постепенно меняется (полный цикл = 20-30 мин реального времени)
- [ ] Закат: тёплые тона, длинные тени
- [ ] Ночь: темно, fog усиливается, видимость снижена
- [ ] Рассвет: постепенное осветление
- [ ] Ночью NPC уходят с рабочих мест (расписание)
- [ ] Магазины закрываются ночью
- [ ] Факел/костёр ночью дают заметный свет

---

## 19. Погода (Этап 8) — если случится во время теста

**Что проверяем:** смена погоды, влияние на геймплей.

- [ ] Погода меняется: ясно — облачно — дождь — гроза
- [ ] Дождь: частицы с камеры, fog меняется
- [ ] Дождь тушит огонь (костёр гаснет)
- [ ] Дождь + земля = грязь (зоны замедления)
- [ ] Гроза: молния может ударить в воду (урон всем в воде)
- [ ] Огонь + сухая трава = распространение пожара

---

## 20. Тренировка оружия (Этап 9) — ~2 мин

**Что проверяем:** тренировочные манекены, мастерство оружия.

- [ ] Найти тренировочный манекен в городе (рядом со стражником / NPC-тренером)
- [ ] Тап по манекену — удар
- [ ] Первые удары неуклюжие (proficiency 0: урон *0.7, скорость *0.6, точность -0.3)
- [ ] С каждым ударом мастерство растёт (скрытое)
- [ ] Через 10-20 ударов — удары быстрее, увереннее
- [ ] Новое оружие (лук, молот) начинается с неуклюжих ударов

## 21. Звук (Этапы 1, 8) — проверять на протяжении всего теста

**Что проверяем:** ambient, шаги, бой, город, музыка бардов.

- [ ] Шаги по разным поверхностям звучат по-разному (трава, камень, грязь, дерево)
- [ ] Ambient природы: птицы днём, сверчки/совы ночью
- [ ] Ambient воды при приближении к реке
- [ ] Звуки боя: удары, рычание волка, фырканье кабана
- [ ] В городе: голоса, стук инструментов, скрип (зонированные)
- [ ] У кузницы громче стук, у таверны — музыка
- [ ] Бард в городе: играет музыку, звук затухает с расстоянием (3D sound)
- [ ] В открытом мире — только ambient, без музыки
- [ ] Микс: ambient не перекрывает шаги и наоборот

## 22. Сон и отдых (Этап 4) — ~2 мин

**Что проверяем:** костёр (пассивный отдых), сон в таверне.

- [ ] У костра: стоять рядом — ускоренное восстановление ран (proximity trigger)
- [ ] Найти таверну в городе — найти кровать
- [ ] Тап по кровати — затемнение ("глаза закрываются") — промотка времени — пробуждение
- [ ] После сна: раны восстановились до порога
- [ ] Время суток сдвинулось (утро)

## 23. Сохранение (Этап 2) — ~2 мин

**Что проверяем:** автосейв и загрузка.

- [ ] Подобрать несколько предметов, переместиться в новое место
- [ ] Свернуть приложение (Home button) — подождать 5-10 сек — вернуться
- [ ] Прогресс на месте (OnApplicationPause сохранил)
- [ ] Убить приложение полностью (swipe up)
- [ ] Запустить заново — персонаж на том же месте, инвентарь сохранён
- [ ] Подобранные предметы не респавнились (состояние ресурсов сохранено)
- [ ] Раны и квесты сохранены

## 24. Респаун ресурсов (Этап 2) — ~1 мин

**Что проверяем:** ресурсы возвращаются через время.

- [ ] Срубить дерево — оно пропадает
- [ ] Подождать (или промотать через сон) — дерево появляется снова
- [ ] Враги тоже респавнятся через время

## 25. Производительность — замерять весь тест

- [ ] FPS стабильный >= 30 (субъективно: нет подтормаживаний)
- [ ] Нет фризов при открытии инвентаря/карты/блокнота
- [ ] Нет фризов при начале боя
- [ ] Нет фризов при входе в город (много NPC и объектов)
- [ ] Нет утечек памяти (приложение не замедляется через 20+ мин)
- [ ] Нет артефактов рендеринга (мерцание, z-fighting)

## Порядок оптимального прогона

Для максимального покрытия за один проход:

1. **Запуск** (п.1) — проверить старт
2. **Движение** (п.2) — базовое управление
3. **Онбординг** (п.3) — подсказки
4. **Подбор предметов** (п.4) — палки, камни, ягоды
5. **Крестьянин** (п.5) — пролог, направление к городу
6. **Инвентарь** (п.6) — лонг-пресс, радиальное меню
7. **Крафт факела и бинта** (п.7) — drag & drop
8. **Костёр** (п.7.3) — разместить дрова, зажечь
9. **Вода** (п.12) — по пути к кабанам
10. **Бой с кабаном** (п.8.1) — получить перелом, лечить шиной
11. **Бой с волком** (п.8.2) — получить кровотечение, лечить бинтом
12. **Еда** (п.9) — съесть ягоды, хил
13. **Дорога к городу** (п.11)
14. **Город, NPC, диалоги** (п.13) — кракозябры, иконки
15. **Торговля** (п.14) — бартер
16. **Кузница** (п.15) — плавка, ковка меча
17. **Квесты** (п.16) — взять, проверить блокнот
18. **Карта** (п.17) — купить, проверить маркеры
19. **Тренировка** (п.20) — манекен
20. **Сон в таверне** (п.22) — ночь, пробуждение
21. **Сохранение** (п.23) — убить приложение, перезапуск
22. **Звук, погода, день/ночь** (п.18, 19, 21) — наблюдать на протяжении всего прогона
